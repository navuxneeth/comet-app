## Troubleshooting

### "Module not found" error?
Run: `pip install -r requirements.txt`

### Port 5001 already in use?
Edit `app.py` and change `port=5001` to another port like `5002`

### Can't access the app?
Make sure you're going to `http://localhost:5001` (not https)
